Dungeons & Dungeons & Dungeons

Java Extensibility Recursion OOP CSV File I/O

Overview: An infinite, text & turn based Java dungeon crawler blending class tactics and race matchups with equipment progression and strategic AP use.

Technical Decisions: Extensively utilizes OOP inheritance to share core fields across characters while subclassing for specific abilities. The dungeon itself is a recursive structure holding lists of enemy objects, allowing combat logic to treat all entities uniformly. Built intentionally for extensibility so new content slots in without refactoring core flows.

Counter Knights: Arcane Offensive

WebGL Unity Raycasting

Overview: Medieval top down 2D tactical shooter with heavy emphasis on recoil. Built with a core focus on recoil-driven spell combat and deterministic movement.

Technical Decisions: Utilises raycasting to determine enemy line-of-sight. Prioritized WebGL stability by avoiding heavy physics engines. Movement is transform-based to keep motion deterministic and recoil predictable. Spells are prefab-driven, allowing independent tuning of damage and spread.
