Minecraft Java Plugins & Server Management

SkyFactions2

Third Party API Integration Custom YAML I/O Event-Driven State Machine Chunk-Based Spatial Logic
  • APIs Used: Vault, WorldEdit, and WorldGuard.
  • Economy & Protection: Hooked into Vault API to handle real-time transactions, and utilised WorldGuard API to generate protected regions for sky-faction land.
  • Dynamic Schematics: Leveraged WorldEdit API to reliably paste starting sky-island schematics into the world upon faction creation.
  • Spatial Mathematics: Programmed chunk-based coordinate math to accurately calculate, validate, and process new land claims.
  • Custom Data Persistence: Wrote custom classes to handle YAML File I/O, persisting faction hierarchies across server restarts.
  • State Management: Built an event-driven state machine to track war declarations, alliance applications, and peace treaties.
  • Damage Interception: Intercepted entity damage events (including projectiles and splash potions) to enforce custom friendly-fire rules based on war states.

WallStreetV2

Bukkit Schedulers Procedural Gen Event Interception 3D Region Math
  • APIs Used: WorldEdit.
  • Asynchronous Tasks: Implemented server-side schedulers to automate non-blocking daily batch processes, such as resetting corporate mines every 24 hours.
  • Procedural Generation: Hooked into the WorldEdit API to define and physically spawn large-scale mine regions, utilising "RandomPattern" to generate the terrain based on exact weighted percentage variables (e.g., 50% stone, 10% diamond).
  • Event Interception & Boundaries: Intercepted player movement and interaction events to dynamically track chunk entry/exit, broadcasting land ownership changes to the UI while calculating if a broken block fell within specific 3D region bounds to enforce build permissions.

Production, Live Operations & its Results

While these specific repositories are archived, the backend architecture principles, including API hooking, asynchronous operations, and spatial mathematics, will continue to inform my current approach to enterprise software development.
